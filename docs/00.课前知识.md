## 做作业 vs 工程开发

### 举例：实现一个除法

- 基本要求：输入二个数字，输出第一个数字除以第二个数字的结果。

- 做作业

- Python版

```python
a = float(input())
b = float(input())
c = a / b
print(c)
```

- C版

```c
#include <stdio.h>

int main()
{
    double a,b,c;

    scanf("%lf",&a);
    scanf("%lf",&b);
    c = a / b;
    printf("%lf\n",c);

    return 0;
}

```

- C++版

```c++
#include <iostream>

int main()
{
    double a,b,c;
    std::cin >> a >> b;
    c = a / b;
    std::cout << c << std::endl;

    return 0;
}
```

```java
import java.util.Scanner;
public class Main
{
	public static void main(String[] args) {
        double a,b,c;
        Scanner sc = new Scanner(System.in);
        a = sc.nextDouble();
        b = sc.nextDouble();
        c = a / b;
		System.out.println(c);
	}
}

```
