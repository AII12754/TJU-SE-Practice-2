## SpringBoot接收参数的8种方式

### 1. 直接把请求参数写在方法的形参中

此场景适用于请求参数较少的情况，任何请求都可以。

```
URL: http://localhost:8080/api/fun1?name=apple&id=123
```

```java
@RequestMapping("/fun1")
public Book fun1(String name, Long id){

    log.info("{}  {}",id,name);
    Book book=new Book(id, name);
    return book;

}
```
